🔍 How It Works

Trap, track & analyze
in four steps

MirrorTrap silently mimics vulnerable services to attract real attackers, then captures everything they do.

Step 01
🪤
Lure
Deploy fake SSH, HTTP and API endpoints that look like real production services. Attackers can't resist.
Step 02
📡
Capture
Every command, credential attempt, HTTP request and payload is silently logged with timestamps and session metadata.
Step 03
🧠
Profile
Gemini AI analyzes each attacker's behaviour — classifying skill level, tools used, intent, and attack type.
Step 04
📊
Visualize
A live SOC-style dashboard shows every attacker, their session timeline, command history and threat intelligence in real time.

⚡ Features

Everything you need to
hunt attackers

🪤
SSH & HTTP Honeypot
Emulates real services including SSH login prompts, web admin panels, and misconfigured APIs to attract opportunistic and targeted attackers.
🧠
Gemini AI Threat Profiling
Each attacker session is analyzed by Google Gemini — classifying attacker type, skill level (Low → Nation-State), tools used, and intent in plain English.
📡
Real-Time SOC Dashboard
Live-updating React dashboard with attacker list, command history, attack timeline, IP geolocation, and auto-refresh — all in a dark SOC aesthetic.
🌍
IP Geolocation & Flags
Resolve attacker IPs to country, city, ISP, and ASN. Country flags displayed inline. Proxy/VPN/Hosting flags are highlighted automatically.
🗄️
SQLite Session Database
Every session, command, and HTTP attack is persisted to a local SQLite database with full query support — no external DB needed.
⏱️
Attack Timeline
Horizontally-scrolling timeline shows every event in chronological order — session start, commands, HTTP scans — color-coded by event type.
